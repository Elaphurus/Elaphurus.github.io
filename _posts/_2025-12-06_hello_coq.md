## 形式化验证 学习笔记

### 环境配置

OCaml + Coq + CompCert + VST

注意查看[这里](https://github.com/PrincetonUniversity/VST/releases)使用匹配的版本

```bash
opam switch create ocaml-4.14 ocaml-base-compiler.4.14.0
eval $(opam env)
opam install coq.8.19.0
opam repo add coq-released https://coq.inria.fr/opam/released
opam install coq-compcert.3.15
opam install coq-vst.2.15
```

在 VSCode 中使用 Coq Proof Assistant IDE：安装扩展 VsCoq Legacy (ID: coq-community.vscoq1)

通过 VST 自带的例子测试配置：

```
cd ~/.opam/coq-8.19/lib/coq/user-contrib/VST/progs64
cp sumarray.c sumarray.v verif_sumarray.v ~/Hello_Coq
cd ~/Hello_Coq
coqc sumarray.v
```
